### Run Locally

To run **CurryCulate** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/CurryCulate.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/CurryCulate.git
```
